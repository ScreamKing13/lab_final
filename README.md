# lab_final
Supported keys:
--version <n> | -v<n>   :  Set the version of something to n <br/>
--something   | -S      :  Does something <br/>
--list <val>  | -l<val> :  Lists n elements if given, ALL otherwise <br/>
--everything  | -E      :  Does EVERYTHING <br/>
--help        | -h      :  Shows help <br/>
